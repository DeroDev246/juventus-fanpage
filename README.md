# ⚫⚪ Juventus FC Fan Page

A full-stack fan website built with **FastAPI** (Python) and Jinja2 templates.

---

## Features

- **Squad Listing** — All players with name, position, age, nationality, stats
- **Player Detail Pages** — Full stats, career highlights, recent match performances
- **Match Schedule** — Upcoming fixtures, past results, live scores tab
- **Live Countdown** — Next match countdown timer
- **Position Filtering** — Filter squad by goalkeeper, defender, midfielder, forward
- **Responsive Design** — Works on mobile and desktop
