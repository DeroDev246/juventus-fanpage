Common Questions

FAQ

How often is player data updated? +
Player stats, ratings and performances are updated manually after each match, usually within 24 hours of full-time.
I found incorrect data — how do I report it? +
Use the contact form above and select "Incorrect Player / Match Data". We'll verify and fix it as soon as possible.
Can I contribute to the project? +
Yes! The project is on GitHub. You can open issues, submit pull requests, or reach out via the form to discuss contributing.
Are live scores actually live? +
The live scores tab refreshes every 30 seconds. During matches, scores are updated manually. We plan to integrate a live API in a future update.
Is this affiliated with Juventus FC? +
No. This is a completely unofficial fan site built by supporters. It has no affiliation with Juventus Football Club S.p.A.
What tech stack does this site use? +
FastAPI (Python) backend, Jinja2 templates, MySQL database with SQLAlchemy ORM, Docker for containerisation, and deployed on Render.
